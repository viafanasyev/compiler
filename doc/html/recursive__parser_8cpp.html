Detailed Description

Implementation of recursive parser.

It parses mathematical expressions using next grammar: 

G = E '\0'
E = T ([+|-] T)*
T = F ([*|/] F)*
F = P (^ P)*
P = '(' E ')' | N | ID | ID '(' E ')'
N = [0-9]+
ID = [a-zA-Z]+

This is an AST building copy of https://github.com/viafanasyev/recursive-parser
