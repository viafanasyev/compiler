staticprotected

This method is created for code like below to be possible (because C++ protected modifier doesn't allow this, unlike in other OOP languages) 

class Parent {
protected:
    virtual void foo() = 0;
};

class Child {
protected:
    void foo() override {
        Parent *p;
        ...
        p->foo();
    }
};

Use dotPrint(child, dotFile) instead of child->dotPrint(dotFile).
